# Краткое описание
Библиотека предназначена для получения различных хеш-сумм в Delphi

# Возможности
- Быстрый (ускорение вплоть до 10-ти раз на asm и до 8-ми раз на delphi) подсчёт crc32, как по прямому, так и по обратному алогоритмам
